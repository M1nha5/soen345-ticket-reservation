# User Stories

- As a customer, I can register with email and phone so that I can reserve tickets.
- As a customer, I can sign in with phone OTP so that I can access my account without email.
- As a customer, I can browse events so that I can select an event.
- As a customer, I can search and filter events by date, location, and category so that I can find relevant events quickly.
- As a customer, I can reserve tickets so that I can attend an event.
- As a customer, I can cancel my reservation so that tickets return to inventory.
- As an organizer, I can register and wait for admin approval so that only validated organizers can publish events.
- As an approved organizer, I can add and edit events so that I can manage offerings.
- As an admin, I can approve or reject organizers so that platform governance is controlled.
- As an admin or organizer, I can cancel an event so that customers are informed of availability changes.
- As a customer, I receive SMS and email confirmations for reserve/cancel actions so that I always have proof of transaction.
